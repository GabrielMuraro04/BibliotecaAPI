# Biblioteca API

Este é um projeto simples de uma API REST para gerenciar livros e usuários de uma biblioteca.  
Estou desenvolvendo ele para praticar ASP.NET Core, Entity Framework Core e MySQL.

## Tecnologias usadas
- .NET 9
- ASP.NET Core
- Entity Framework Core
- MySQL (com Pomelo)
- Swagger para testar os endpoints

## Funcionalidades atuais
- CRUD de Livros (criar, listar, editar, excluir)
- Configuração do banco via EF Core Migrations

- Validações
